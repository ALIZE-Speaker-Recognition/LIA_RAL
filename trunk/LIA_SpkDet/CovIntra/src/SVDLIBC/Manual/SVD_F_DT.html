SVD_F_DT

Dense Text Matrix File Format

Format:

numRows numCols
for each row:
  for each column:
    value

Newlines and spaces are equivalent.

Example:

4 3
2.3  0  4.2
0   1.3 2.2
3.8  0  0.5
0    0   0
