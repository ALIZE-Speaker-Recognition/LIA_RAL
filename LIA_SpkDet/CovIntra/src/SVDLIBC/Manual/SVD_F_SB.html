SVD_F_SB

Sparse Binary Matrix File Format

Format:

numRows numCols totalNonZeroValues
for each column:
  numNonZeroValues
  for each non-zero value in the column:
    rowIndex value

All values are 4-byte integers except value, which is a 4-byte float. All are in network byte order.
